# PingPong

## Descrição do Projeto

Este projeto é um jogo de ping pong desenvolvido utilizando 100% JavaScript. O objetivo é proporcionar uma experiência interativa e divertida diretamente no navegador, sem a necessidade de plugins ou instalações adicionais.

## Funcionalidades

- **Jogo de Ping Pong Clássico**: Controle a raquete para rebater a bola e desafie a pontuação máxima.
- **Interface Simples e Intuitiva**: Design minimalista focado na jogabilidade.
- **Responsividade**: Compatível com diferentes tamanhos de tela e dispositivos.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
